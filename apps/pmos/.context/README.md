# PMOS Context Directory

This directory contains auto-generated context files for AI-assisted development.

## Files

- `runtime-context.md` — Generated by `npx tsx scripts/build-pmos-context.ts`

## Usage

### Generate fresh context
```bash
# Requires PMOS running on port 3200
npx tsx scripts/build-pmos-context.ts
```

### Inject into Copilot / ChatGPT
Copy the contents of `runtime-context.md` into your prompt prefix.

### Future: automatic injection
The structure is ready for automatic injection via:
- VS Code workspace instructions (`.github/copilot-instructions.md`)
- Custom Copilot agent attachment
- Prompt template prefix

## Philosophy
PMOS does not give AI access to everything.
PMOS delivers the relevant working set for the current development focus.
